 Description of Measurements:
 The accompanying matlab files contain radial vibration measurements on the bearing housing of the
 SpectrQuest Machinery Fault Simulator test rig http://spectraquest.com/ 
 The set of measurements contain two files, all with the following measurement parameters, with a known
 inner and outer race bearing fault.

 Sampling frequency		:	51200 Hz (useful bandwidth 20 kHz)
 Length of record		:	10 seconds
 Channel_1			:	Radial Bearing Housing Acceleration (m/s^2)
 Channel_2			:	Tacho - once per revolution pulse (Volts)
 Rotational speed 		:	29 Hz

 Bearing Information
 Bearing No.			:	MB ER-16K
 Number of balls		:	9
 Ball Diameter			:	7.9375mm
 Pitch Diameter			:	38.50mm
 FTF				:	0.3969 x (shaft speed)
 BPFO				:	3.572 x (shaft speed)
 BPFI				:	5.428  x (shaft speed)
 BSF				:	2.32 x (shaft speed)
